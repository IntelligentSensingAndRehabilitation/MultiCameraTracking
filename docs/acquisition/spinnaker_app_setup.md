# Acquisition Software Installation Guide

To install Spinnaker apps locally (like **SpinView**):
- Download the latest Spinnaker SDK from the FLIR website https://www.flir.com/support-center/iis/machine-vision/downloads/spinnaker-sdk-download/spinnaker-sdk--download-files/
    - Unzip the object
        - tar -xzf archive.tar.gz
    - Look at the README and install the prerequisite required dependencies
    - Go into the unzipped directory and install spinnaker
        - sudo sh install_spinnaker.sh
    - There may be a pop-up that you need to accept, use **TAB** to navigate to <OK>and press **ENTER**
